This is an advanced To-Do List web application developed using HTML, CSS, and JavaScript. The application helps users efficiently manage their daily tasks by providing features such as adding, editing, deleting, and marking tasks as completed.

It includes advanced functionalities like task filtering (All, Active, Completed), a task counter to track progress, and an option to clear completed tasks. The application also uses browser local storage to save tasks, ensuring that data is not lost even after refreshing the page.

The user interface is designed to be simple, clean, and user-friendly, making task management easy and effective. This project demonstrates key front-end development concepts such as DOM manipulation, event handling, and data persistence using local storage.
